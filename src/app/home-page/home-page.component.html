Welcome to Netex Material UI Library

Netex Material UI Library is a library of components based on Angular Material framework. This library complements the Angular Material library with our own styles guide and new customized components. This project provides a set of reusable, well-tested, and accessible UI components based on Google's Material Design Specifications.

Netex Material UI Library is a mature and stable product that is ready for production use. Developers should note that Netex Material UI Library works only with Angular 7.
