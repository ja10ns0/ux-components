Forms

All of the attributes that can be used in a form will be part of ntx-form.

Forms types

These are all the types of nxt-list: ntx-form-input, ntx-form-textarea, ntx-form-datepicker, ntx-form-checkbox, ntx-form-checkbox-circle, ntx-form-radiobutton y ntx-form-slidetoggle.

Configuration

The configuration of each ntx-form is being shown in its own .ts file.
