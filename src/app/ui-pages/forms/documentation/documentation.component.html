Forms

All of the attributes that can be used in a form will be part of ntx-form.

Forms types

These are all the types of nxt-list: ntx-form-input, ntx-form-input-no-label, ntx-form-textarea, ntx-form-select, ntx-form-select-no-border, ntx-form-select-button, ntx-form-select-border-button, ntx-form-select-thin, ntx-form-datepicker, ntx-form-checkbox, ntx-form-checkbox-circle, ntx-form-radiobutton, ntx-form-slidetoggle, ntx-form-slider, ntx-form-searchbox y ntx-form-searchbox-background.

Configuration

The configuration of each ntx-form is being shown in its own .ts file.
