Margins Helper Classes

Netex Material UI Library utiliza una guia de estilos propia que nos permite customizar nuestros componentes. Esta guia de estilos cuenta con unas clases css "helpers" con los estilos más usados que te ayudará a maquetar tus componentes de manera más sencilla.

En esta página te mostramos las clases de márgenes que te permiten añadir separación entre los distintos elementos de la página.
