Release v4.0.0: Error pages, progress spinner and more

29 abril 2019

What’s new section is our monthly update that highlights recent product improvements we’ve made so you can easily stay up to date on what’s new.

What’s new

Empty pages

Arrancamos la nueva versión de la librería con el nuevo componente ntx-empty-page para poner en nuestras pantallas vacías e indicarle al usuario que aún no hay contenido en esa pantalla.

Error pages

Nuestra siguiente novedad son las necesitadas páginas de error. Estos son los componentes:

  • Error page 403 — Las páginas de error 403 se representarán con el componente ntx-error403
  • Error page 404 — Las páginas de error 404 se representarán con el componente ntx-error404
  • Error page 500 — Las páginas de error 500 se representarán con el componente ntx-error500
  • Error page 503 — Las páginas de error 503 se representarán con el componente ntx-error503

New icons

Con esta actualización sacamos un nuevo lote de iconos. Para ello se han tocado varios archivos css que debeis actualizar:

  • ux-components/src/styles/app/utils.scss
  • ux-components/src/styles/app/variables.scss
  • ux-components/src/styles/fonts/styles.scss
  • ux-components/src/styles/fonts/variables.scss
  • ux-components/src/assets/fonts/icons/netex.eot
  • ux-components/src/assets/fonts/icons/netex.svg
  • ux-components/src/assets/fonts/icons/netex.ttf
  • ux-components/src/assets/fonts/icons/netex.woff

Trees

En esta versión aportamos nueva documentación de qué es, para qué sirve y cómo se configura el componente ntx-tree

Chips

Estrenamos un nuevo modelo de chip ntx-image-subtitle-chip con imagen, título y subtítulo para aportar la información más relevante a simple vista.

Progress spinner

Ya tenemos en la librería el componente ntx-progress-spinner para indicar a nuestros usuarios que hay un proceso de carga en curso.

Card 14

Lanzamos una nueva versión de cards, el componente ntx-card14 que es una versión de la card8 que indica que esa card ya ha sido añadida.
