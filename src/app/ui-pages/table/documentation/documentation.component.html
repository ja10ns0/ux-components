Basic table

ntx-basic-table is a component based on Angular data-table that can be used to display rows of data.

Configuration

The content of the table is being shown in its own .ts file.

Also you can customize ntx-basic-table with a flat design through @Input property like this: <ntx-basic-table [style]='flat'>

This 'flat design' hides paginator, removes border shadow and background color.
