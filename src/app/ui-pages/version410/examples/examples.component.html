Release v4.1.0: Inputs with autocomplete, new list designs, bug fixed and more...

dia mes 2019

What’s new section is our monthly update that highlights recent product improvements we’ve made so you can easily stay up to date on what’s new.

What’s new

New icons

With this update we have a new batch of icons. For this, several css files have been touched, which you must update to see the icons correctly:

  • ux-components/src/styles/app/utils.scss
  • ux-components/src/styles/app/variables.scss
  • ux-components/src/styles/fonts/styles.scss
  • ux-components/src/styles/fonts/variables.scss
  • ux-components/src/assets/fonts/icons/netex.eot
  • ux-components/src/assets/fonts/icons/netex.svg
  • ux-components/src/assets/fonts/icons/netex.ttf
  • ux-components/src/assets/fonts/icons/netex.woff

NOTE: We have deleted icon-speacker and added icon-speaker instead, for an spelling error.

Form input autocomplete

We released a new type of input, ntx-form-input-autocomplete enhanced by a panel of suggested options, which are filtered as they are written on it.

Autocomplete chip

We launched a new version of chips, ntx-autocomplete-chip integrated with an autocomplete input that suggests options as you type.

Two lines vertical list with avatar

With this update we have a new type of list, ntx-list-two-lines-avatar-vertical that vertically integrates the image of the avatar with the text.

List horizontal with icons

We release a new type of list, ntx-list-horizontal-icon that integrates icons in the horizontal lists.

Entity detail panel

Our next innovation is a detail panel, app-entity-detail-panel, which shows us the detail of an entity in the header of a panel.

Simple options menu

In this version we launched a new type of menu, ntx-menu-options-simple much simpler than the previous ones, without icons, buttons or space bars.

Dashboard menu

In this version we launched a new type of menu, ntx-menu-toolbar-dashboard which shows a grid of elements.
