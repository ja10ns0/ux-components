Lists

ntx-list is a component that wraps and formats a series of line items.

Trees types

There are three types of nxt-list: use ntx-list-two-lines-avatar for lists that need have an image and two lines of content: title and subtitle. Use ntx-list-two-multilines-icon for lists that represent sections, where the elements to show are its image, title and an indeterminate number of links to the different subsections. And use ntx-list-one-line-options for lists that have a loading process and options.

Configuration

The content of each list is being shown in its own .ts file, feel free to move this content to a external json file.
