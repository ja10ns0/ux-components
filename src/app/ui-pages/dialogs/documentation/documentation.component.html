Dialogs

ntx-dialog is a component that can be used to open modal dialogs. This dialogs inform users about a task and can contain critical information, require decisions, or involve multiple tasks..

Dialogs types

There are three types of nxt-dialog: use ntx-simple-dialog for dialogs that need a confirmation or inform about something relevant, use ntx-warning-dialog to inform the user that an error has occurred, and use ntx-proccess-dialog when a proccess is going to begin.

Configuration

Each dialog is composed of two components. The first component contains the button that calls the dialog, and the second component contains the content of the dialog.
