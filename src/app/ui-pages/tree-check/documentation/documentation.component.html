Tree check

ntx-tree-with-checks is a component that can be used to display hierarchy data. These trees use the nested structure o organize the content hierarchically.

Trees types

There are two types of nxt-trees: use ntx-tree-with-checks for trees that need to be able to select their content to perform different actions.

Configuration

Each tree has its own configuration in the .ts file. The tree data is being shown in this file but feel free to move this content to a external json file.
