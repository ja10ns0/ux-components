Cards

ntx-card is a content container for text, photos, and actions in the context of a single subject.

Card 11

Use ntx-card11 if you need a group of selectable cards by radiobuttons and have a title, a description, a counter number and a progress bar.

Configuration

In .ts file of each component, we have added a click event to show the style of the cards when they are selected.

Card11 v4.0.1

  • One-way data binding from input element's data property.
  • Bind ngClass directive to element's selected property.
  • Define an output(@Output) element's radiobuttonChange property.
