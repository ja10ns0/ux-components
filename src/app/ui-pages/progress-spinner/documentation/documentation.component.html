Progress spinner

ntx-progress-spinner is a circular indicator of progress and activity.

Configuration

Color attribute can be configured in each progress spinner, just by adding the atribute [color] in the html tag. In .ts file we have added this variable as strings so it can be configured.

By default, the color should be primary color, and can be used by specifying it in color atribute [color]="'primary'".

The diameter and the stroke's width of the progress spinner are shown by an atribute in the html. The default values must be those established in the component. Feel free to change them if the designer requires it.
