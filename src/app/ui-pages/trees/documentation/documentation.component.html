Trees

ntx-tree is a component that can be used to display hierarchy data. These trees use the nested structure o organize the content hierarchically.

Trees types

There are two types of nxt-trees: use ntx-tree for trees that need to have their content organized hierarchically and their content is selectable to be consulted one by one

Configuration

Each tree has its own configuration in the .ts file. The tree data is being shown in this file but feel free to move this content to a external json file.
