Icons

mat-icon directive makes it easier to use vector-based icons in your app.

Icon font

Netex has built it's own icon font that allow us to use a font where the glyphs in the font are our icons instead of text. Every single icon in your app has to use this font.

Configuration

To use this font, you need to add icon-netex class and also you have to add the name of the icon you need, for example icon-heart. Therefore, you can use specific classes that make you easier to change icon sizes: icon-small, icon-medium or icon-big. Your element classes will be something like this: class="icon-netex icon-heart icon-medium".
